Algumas anotações de códigos para git:
Para Windows - dir: lista os diretórios contidos na pasta onde você está situado
para linux - ls: mesmo objetivo descrito acima

cd windows/ - vai para a pasta windows, por exemplo
cd .. - retrocede 1 nível na navegação

Para windows- cls - limpa a tela 
para linux - clear

mkdir - criar uma pasta

git init - inicializa um repositório vazio no git
git commit - adiciona strings
